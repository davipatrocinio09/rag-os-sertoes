# RAG - Os Sertões

Projeto de Retrieval-Augmented Generation (RAG) utilizando:

- Python
- LangChain
- OpenAI
- FAISS
- HuggingFace Embeddings

O sistema carrega o livro "Os Sertões", cria embeddings vetoriais e responde perguntas usando busca semântica.

## Tecnologias

- Python
- LangChain
- FAISS
- GPT-4o-mini
- Sentence Transformers

## Exemplo

Pergunta:

Quem foi Antônio Conselheiro?

Resposta:

Líder religioso e social que fundou a comunidade de Canudos.
